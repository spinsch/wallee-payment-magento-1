1Prerequisites

If you don’t already have one, create a wallee account and set up a space.

We recommend to use the wallee setup assistant for configuration. If you prefer to do the setup manually, follow these steps:

  1. Create an application user that is allowed to access the space you want to link the Magento store to. Navigate to your wallee account → application user and create your application user. The user ID and the authentication key will be shown. Make sure to write this information down.

  2. Make sure that you grant the application user the necessary rights (Account Admin) in order for the Magento plugin to be able to set up the necessary webhooks, payment methods etc.

  3. Set up at least one processor, one payment method and a connector configuration. More Information about the processor concept can be found in the wallee documentation.

If your are using Magento 1.6.x you need to install the Magento compatibility module.

2Installation

  1. Upload the extension files to your store’s root directory using FTP/SSH.

  2. Login to the backend of your Magento store.

  3. Clear the cache in System → Cache Management.

  4. Log out of the backend, and log back in again.

3Configuration

  1. Navigate to System → Configuration → wallee Payment in your Magento store’s backend and enter the wallee application user’s access information that you created before.

  1. You need to switch the configuration scope to Main Website to enter the space id.

  1. After saving the configuration, the payment methods are created in the Magento store and the necessary webhooks are set up.

4Payment Method Configuration

Most of the payment method configuration is done in wallee. After following the steps of the previous steps, the wallee payment method configurations will be synchronized automatically into the Magento store. There are just a few payment method settings in the Magento store in System → Configuration → Payment Methods. To see the wallee payment methods, you need to switch the configuration scope to Main Website.

5Third party support

These third party checkout extensions are supported by the wallee Magento extension:

  • Aheadworks One Step Checkout

  • GoMage LightCheckout

  • OneStepCheckout

  • IWD Checkout Suite

  • IWD One Page / Step Checkout

  • Magestore One Step Checkout

  • TemplatesMaster One Page Checkout

6FAQ

6.1The payment methods are not showing up.

Make sure that you followed configuration step 2 by stating your wallee space ID in the Magento backend. By saving the configuration form the synchronization of the payment methods and the set up of the webhooks are initiated.

If this does not solve the problem, it could be that you use a special fee or coupon module that we do not support. Try to disable this plugin and see if it helps. The payment methods are only displayed if the plugin’s total calculation matches the actual order total.

6.2How can I change the look and feel, payment method name, emails etc.?

All those configurations are done directly in wallee. For more information have a look at the wallee documentation.
